🚀 SaveMoney: Sistema de Gestão Financeira PessoalStatus do Projeto

Em Desenvolvimento (Metodologia Scrum) - TCC UniCesumar 2026.Descrição do Projeto

O SaveMoney é uma plataforma web de controle financeiro pessoal, gratuita, intuitiva e unificada, desenvolvida como Trabalho de Conclusão de Curso (TCC) para mitigar o elevado endividamento e a carência de educação financeira no Brasil. Nosso objetivo é democratizar o acesso a ferramentas de gestão financeira de qualidade, combatendo o estresse financeiro e promovendo o empoderamento econômico.

A plataforma vai além do simples registro de transações, fornecendo uma visão analítica e estratégica das finanças pessoais, incluindo:

    Visualização clara de padrões de gastos.
    Projeção de orçamentos.
    Estabelecimento de metas financeiras realistas.
    Fornecimento de insights práticos.

O Problema que Endereçamos

    Endividamento Elevado: 78,9% das famílias brasileiras estão endividadas (PEIC, julho/2024).
    Carência de Educação Financeira: 45% dos adolescentes brasileiros (15 anos) têm baixo desempenho em letramento financeiro (PISA/OCDE).
    Barreira de Acesso: A maioria das ferramentas de controle financeiro no mercado é paga ou possui versões gratuitas com recursos limitados ou complexos, excluindo a população mais vulnerável.

ObjetivosObjetivo Geral:
Desenvolver uma plataforma de controle financeiro pessoal, unificada e intuitiva, que seja acessível e contribua para a educação financeira, a redução do endividamento e a ansiedade popular.

Objetivos Específicos:

    Promover educação financeira inclusiva.
    Reduzir o endividamento e a inadimplência.
    Reduzir a ansiedade causada pela falta de planejamento financeiro.
    Apoiar pequenos empreendedores na separação das finanças pessoais e empresariais.

Metodologia de Desenvolvimento:

O sistema está sendo desenvolvido com base no framework ágil Scrum, que permite entregas incrementais e adaptações contínuas, garantindo a rápida resposta às necessidades dos usuários e a qualidade do produto final.Arquitetura e Modelagem do Sistema

O projeto é estruturado em módulos funcionais e utiliza as seguintes técnicas de modelagem:

    Módulos Funcionais: Autenticação, Lançamentos de Receitas e Despesas, Definição de Metas e Dashboards Visuais.
    Modelagem Utilizada: Diagramas de Casos de Uso, Diagrama de Classes e Diagrama Entidade-Relacionamento (DER).
    Documentação Adicional: Dicionário de Dados e User Stories (Histórias de Usuário) para detalhamento dos requisitos.
